# sceneflowproto  
**Scene types**- title, credits, game levels, menu, map, settings  
**communication** - level unlocking system  
**reachability** -  scene switch case:  
case '0': logo  
case '1': cred  
case '2': title  
case '3': map  
case '4': settings  
case '5': pause  
case '6': level1  
case '7': level2  
case '8': level3  
**transitions** - we have fade in intro, title, and map
